# tna-tools

개발 워크플로우 자동화를 위한 Claude Code 플러그인.

## Rules

- 모든 사용자 커뮤니케이션은 한국어로 한다.
- API 토큰이나 인증 정보를 설정 파일이나 로그에 저장하지 않는다.
- 회의록 내용을 로컬 파일이나 캐시에 저장하지 않는다.
- 사용자 확인 없이 외부 채널(슬랙, poke)로 메시지를 전송하지 않는다 (--auto 제외).
- poke 메시지에는 구체적인 금액, 개인정보, 기밀 사항을 포함하지 않는다.
- Confluence 페이지는 항상 새로 생성한다 (기존 페이지를 덮어쓰지 않음).
- MCP 세션 만료 시 재인증 안내를 표시하고 실행을 중단한다.
- 스킬이나 설정을 변경하면 반드시 `plugin.json`과 `marketplace.json`의 버전을 함께 올린다 (캐시 무효화 필수).
