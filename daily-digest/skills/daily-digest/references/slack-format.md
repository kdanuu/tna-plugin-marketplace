# 슬랙 DM 리포트 포맷 가이드

## 메시지 구조

```
📋 오늘의 회의 리포트 ({YYYY-MM-DD})
━━━━━━━━━━━━━━━━━━━━

🗓 회의 {N}건

1. [{회의 제목}] {시작시간}-{종료시간} ({소스: caret/Google Meet})
   참석: {참석자 목록}
   요약: {2-3문장 핵심 요약}
   액션 아이템:
   • @{담당자} — {할 일} ({마감일})
   • @{담당자} — {할 일} ({마감일})

2. [{회의 제목}] {시작시간}-{종료시간} ({소스})
   ...

💡 오늘의 핵심
• 총 액션 아이템: {N}건 (내 담당: {M}건)
• 다음 회의: {다음 회의 정보}
```

## 작성 규칙

- 회의는 시간순으로 정렬한다
- 소스 표시: 각 회의 옆에 `(caret)` 또는 `(Google Meet)` 표시
- 참석자: 이름만 나열 (이메일 제외)
- 요약: 핵심 결정사항 위주, 2-3문장 이내
- 액션 아이템: 담당자 + 할 일 + 마감일 형식
- "오늘의 핵심" 섹션: 내 담당 액션 아이템 수와 다음 일정 강조
- 슬랙 mrkdwn 문법 사용 (*bold*, _italic_, `code`)
- 회의 0건일 때는 이 포맷을 사용하지 않음 (스킬에서 조기 종료)
