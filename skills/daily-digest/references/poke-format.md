# poke 문자 비서 스타일 포맷 가이드

## 톤

- 간결한 비서 톤, 존댓말
- 핵심만 전달, 불필요한 수식 제거
- 정보 밀도 높게

## 구조

```
오늘 회의 {N}건 마무리됐습니다.
내 액션 아이템 {M}건: {항목1}({마감일}), {항목2}({마감일}).
{다음 일정 정보}.
```

## 규칙

- 200자 이내 권장 (문자 채널 특성)
- 회의 제목은 생략, 건수만 표시
- 내 액션 아이템만 핵심적으로 나열
- 다음 일정이 있으면 마지막 줄에 포함
- 액션 아이템이 3건 초과 시: "내 액션 아이템 5건: 기술 스펙 초안(3/25) 외 4건"

## 민감 정보 제외

문자 채널 특성상 보안 취약하므로 아래 항목은 절대 포함하지 않는다:

- 구체적인 금액, 매출, 비용 수치
- 개인정보 (전화번호, 이메일, 주소)
- 기밀 사항 (인사 변동, 미공개 계약 등)
- 비밀번호, 토큰, API 키
